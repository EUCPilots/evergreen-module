# Get-EvergreenAppsPath

## SYNOPSIS

Returns the local cache path for the Evergreen apps functions and manifests.

## SYNTAX

```
Get-EvergreenAppsPath
```

## DESCRIPTION

Returns the local cache path for the Evergreen apps functions and manifests.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-EvergreenAppsPath
```

Description:
Returns the local cache path for the Evergreen apps functions and manifests.
